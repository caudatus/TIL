# SwipeCellKit
Swipeable UITableViewCell based on the stock Mail.app, implemented in Swift.

- 왼쪽, 오른쪽 스와이프 액션
- 액션 버튼 : text only, text + image, image only
- 햅틱
- 수정 가능 : Border, Drag, and Reveal
- 스와이프 중 액션 버튼 수정 가능

## Requirements
- Swift 4.1
- Xcode 9+
- iOS 9.0+
